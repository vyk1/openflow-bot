Documentação da Estrutura JSON para Respostas do Chatbot DRICA

Este documento descreve a estrutura JSON utilizada para definir as intenções e as respostas que o chatbot DRICA irá apresentar aos usuários. A DRICA tem como foco auxiliar na prevenção e orientação sobre violência no trabalho no setor público.

Estrutura Geral

A estrutura JSON segue uma organização hierárquica em que as intenções (intents) são os principais nós de diálogo, e dentro de cada uma, temos mensagens de fala (speech), conteúdos (content), e opções de seguimento (followUps). Cada intenção pode ter diversas ramificações baseadas nas escolhas feitas pelo usuário.

Intenções

Cada intenção corresponde a um tópico específico ou uma pergunta a ser tratada pelo chatbot. As intenções estão divididas em categorias com os seguintes campos:

  • name: O nome da intenção (para fins de identificação).
  • speech: Um array de objetos que contém a resposta textual ou visual do chatbot.
  • content: Define as opções que o usuário pode escolher e outros tipos de conteúdo complementar.
  • followUps: As intenções de seguimento, que especificam as respostas e opções subsequentes dependendo da escolha do usuário.

Saudação

Descrição:

Esta intenção serve como introdução, onde a DRICA se apresenta e pergunta ao usuário se ele sabe o que é um "bot".

"saudacao": {
    "name": "Saudação",
    "speech": [
        {
            "type": "info",
            "title": "Olá! Eu sou a DRICA, assistente da DIC para prevenir e tirar dúvidas sobre violência do trabalho no setor",
            "subtitle": "Meu objetivo é ajudar você a entender melhor o que é violência no trabalho e como preveni-la. Vamos começar?"
        }
    ],
    "content": [
        [
            {
                "title": "Me diga, você sabe o que é um \"bot\"?",
                "type": "info"
            }
        ],
        {
            "options": [
                {
                    "text": "Sei o que é um bot",
                    "followUp": "sei-bot"
                },
                {
                    "text": "Não sei o que é um bot",
                    "followUp": "nao-sei-bot"
                }
            ]
        }
    ],
    "followUps": { ... }
}

Campos:

  • speech: Uma mensagem de boas-vindas é fornecida, explicando o papel do chatbot.
  • content: Uma pergunta é feita ao usuário, seguida de duas opções para continuar:
    • "Sei o que é um bot"
    • "Não sei o que é um bot"

Ramificações:

  • sei-bot: Se o usuário souber o que é um bot, a DRICA confirma e oferece a opção de continuar.
  • nao-sei-bot: Se o usuário não souber, a DRICA explica o conceito de bot e oferece a opção de prosseguir.

Violência no Trabalho

Descrição:

Nesta intenção, a DRICA introduz o conceito de violência no ambiente de trabalho, destacando que ela pode ir além da violência física.

"violencia-trabalho": {
    "name": "Violência no Trabalho",
    "speech": [
        {
            "type": "info",
            "title": "Você sabia que violência no trabalho, não é só fisica?",
            "subtitle": "O Manual de Saúde e Segurança do Servidor Público de Santa Catarina define a violência no ambiente de trabalho como toda e qualquer ação ou comportamento, explícito ou sutil, que resulte em agressão, humilhação ou ameaça ao servidor ou aos indivíduos relacionados a ele."
        }
    ],
    "content": [
        [
            {
                "type": "link",
                "title": "Programa Combate ao Assédio da CGE SC",
                "link": "https://www.cge.sc.gov.br/combateaoassedio/"
            }
        ],
        {
            "options": [
                {
                    "text": "Mais sobre discriminação",
                    "followUp": "violencia-discriminacao"
                },
                {
                    "text": "Mais sobre assédio",
                    "followUp": "violencia-assedio"
                }
            ]
        }
    ],
    "followUps": { ... }
}

Campos:

  • speech: Fornece uma explicação inicial sobre violência no trabalho.
  • content: Oferece um link para o "Programa Combate ao Assédio da CGE SC" e opções de seguimento:
    • "Mais sobre discriminação"
    • "Mais sobre assédio"

Ramificações:

  • violencia-discriminacao: Fornece mais informações sobre discriminação.
  • violencia-assedio: Explica mais sobre os diferentes tipos de assédio.

Ramificações Específicas

Cada followUp define novas ramificações e fluxos de conversa, permitindo que o chatbot se aprofunde em cada tema conforme a escolha do usuário. Por exemplo, dentro de "violência no trabalho", temos:

  • violencia-discriminacao: Aborda o conceito de discriminação no trabalho.
  • violencia-assedio: Descreve os diferentes tipos de assédio no ambiente de trabalho, como assédio moral e sexual, oferecendo opções para o usuário escolher e aprender mais.

Tipos de Mensagem

As mensagens que o chatbot pode apresentar ao usuário estão organizadas da seguinte forma:

  • info: Informação textual com título e subtítulo, usada para explicar conceitos.
  • image: Imagens que podem ser utilizadas para ilustrar informações.
  • link: Links para recursos externos que podem ser relevantes para o usuário.
  • options: Botões interativos que permitem ao usuário selecionar a próxima ação.

Exemplo de FollowUp: "O que é um bot?"

Quando o usuário escolhe a opção "Não sei o que é um bot", a DRICA explica o conceito e oferece uma imagem e um botão para continuar.

"nao-sei-bot": {
    "speech": [
        {
            "type": "image",
            "src": "https://encrypted-tbn0.gstatic.com/images?q\u003dtbn:ANd9GcStEBHnnElM5xC5XAGiXerMjzMjpOhycZD8YQ\u0026s",
            "title": "O que é um bot?"
        },
        {
            "type": "info",
            "title": "Um bot é um programa de computador que simula ações humanas repetidas vezes de maneira padrão."
        }
    ],
    "content": [
        [],
        {
            "options": [
                {
                    "text": "Continuar",
                    "type": "end",
                    "followUp": "violencia-trabalho"
                }
            ]
        }
    ]
}
